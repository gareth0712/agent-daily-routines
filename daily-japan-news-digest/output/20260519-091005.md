*CLAUDE.md / Memory / SKILLS / HOOK / settings.json を体系的に解説*
來源：Zenn - Claude Code｜發布：2026-05-18 22:44 JST

Claude Code の振る舞いを制御する「ハーネス機能」として CLAUDE.md（プロジェクト指示）、Memory（セッション持続情報）、SKILLS（再利用可能命令テンプレート）、HOOK（イベントトリガー処理）、settings.json（権限・モデル設定）が体系的に解説されている。各機能の役割分担を理解することで、単なるプロンプト改善を超えたシステマティックな AI 開発が可能になる。特に HOOK は SessionStart・PostToolUse・Stop などのイベントに対応し、外部コマンドを自動実行するための強力な仕組み。SKILLS は `/skill-name` の形式でセッション中に呼び出せる命令テンプレート。settings.json の `allowedTools`・`blockedTools` 設定でセキュリティ境界も制御できる。
