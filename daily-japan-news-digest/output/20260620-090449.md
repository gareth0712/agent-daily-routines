### [Claude Code Dynamic Workflows 完全ガイド](https://zenn.dev/mdtechknowledge/articles/claude-code-dynamic-workflows-guide-2)
*Claude Code Dynamic Workflows 完全指南*
來源：Zenn - Claude Code｜發布：2026-06-19 20:20 JST

Dynamic Workflows は JavaScript オーケストレーションシステムで、複数のサブエージェントを自動調整するスクリプトを生成し、コンテキストウィンドウを拡張せずに大規模タスク自動化を実現する。起動方法は 3 種類：単発タスク向け `ultracode` キーワード、セッション全体の自動化を有効にする `/effort ultracode`、`/deep-research` などの事前定義コマンドがある。**v2.1.172** では階層的エージェントスポーニングが導入され、サブエージェントが子エージェントを生成できる最大 **5 段ネスト**が可能となった。同時実行は最大 **16 エージェント**、総計 **1,000 エージェント/実行**を上限とし、実際の並列度はプランではなくローカル CPU コアに依存する。`/workflows` コマンドでフェーズ別の進捗・累積トークン・実行時間を監視でき、一時停止・停止・再利用可能なコマンドとして保存が可能。**v2.1.160+** ではシェル起動ファイルやビルド設定編集に安全制限が追加されており、無人実行時のリスクが軽減された。
