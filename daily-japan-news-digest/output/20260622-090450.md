### [Claude Code のブラウザ自動化を全部試した — 最終結論](https://zenn.dev/davidai311/articles/browser-tools-evolution-verdict)

來源：Zenn - Claude Code｜發布：2026-06-21 20:54 JST

作者在 2026 年 2-3 月系統測試了六款 Claude Code 瀏覽器自動化工具。**PinchTab** 以 Chrome 無障礙樹分析為核心，僅需約 800 token/頁，效率最高，適合日常讀取與測試；**browser-use**（Python 框架）在複雜多步驟表單自動化中表現最佳；**agent-browser**（Rust 製，Vercel Labs）比 MCP 方案減少 93% token 消耗（3,000–5,000 token/頁）。
