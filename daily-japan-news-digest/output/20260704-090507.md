### [Get-Content -RawがJSONを375KBに爆発させる話](https://zenn.dev/idonthaveapen/articles/get-content-raw-json-explosion)
*PowerShell の Get-Content -Raw が JSON を 375KB に膨張させる問題*
來源：Zenn - Claude Code｜發布：2026-07-04 00:13 JST

Windows 環境で Claude Code フックを使用する際、PowerShell の `Get-Content -Raw` コマンドが JSON ファイルを数 KB から 375KB に膨張させるバグを詳細調査。原因は `Get-Content -Raw` が UTF-8 BOM を付加しつつ改行コードを変換する際の文字エンコーディング問題。⚠️ Windows で Claude Code フックを利用しているユーザーは JSON 処理パイプラインを確認すること。回避策として `[System.IO.File]::ReadAllText()` や `Get-Content -Encoding UTF8NoBOM` の使用を推奨。ファイルサイズ肥大化はトークン消費増加・コスト増大・レート制限にも直結するため早急な対応が望ましい。
