來源：Zenn - Claude Code｜發布：2026-07-18 08:00 JST

- 新功能 **MCP Connector 對應**：Artifacts 現在可直接呼叫 MCP Connector，頁面每次載入時從外部資料來源取得最新資料，不再是發布當下的靜態快照。
- **版本要求**：需 Claude Code **v2.1.209+**；本機 MCP Server 僅在 Artifact 建立時可用，後續訪客無法呼叫。

- ⚠️ 限制：Artifacts 受嚴格 CSP 限制，外部 fetch 全部封鎖；MCP Connector 是唯一合法的資料取得途徑。公開分享的 Artifact 目前**不支援**此功能（需要認證）。

*滿足了清單，但沒有達成目的——AI Agent 運用的盲點*
來源：Zenn - Claude Code｜發布：2026-07-18 06:31 JST

- 核心教訓：AI Agent 可能無批判地將未驗證的文件聲明納入後續報告，在問題顯現前已跨系統放大錯誤。
